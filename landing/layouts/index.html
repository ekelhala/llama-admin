llama-admin

A small Go server you deploy on your inference host. Spawn and supervise llama-server processes, expose an OpenAI-compatible API protected by API keys, and manage everything from a CLI that authenticates users via OAuth.

Get started Download

What it does

  • Spawns and supervises one or more llama-server instances
  • Exposes an OpenAI-compatible API (/v1/*) protected by API keys, routing by request model field
  • Management API (/api/v1/*) protected by OAuth-issued session tokens
  • CLI authenticates via OAuth device flow; manages instances, models and API keys

Install

Server (on the inference host):

curl -fsSL {{ .Site.Params.repo }}/releases/download/latest/install-server.sh | sh

CLI (on your workstation):

curl -fsSL {{ .Site.Params.repo }}/releases/download/latest/install-cli.sh | sh

Then point it at your server and log in:

llama-admin config set-server https://your-host:8080
llama-admin auth login

Or build from source with make build.
